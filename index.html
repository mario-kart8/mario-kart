Robô Seguidor de Linha

Um projeto acadêmico de automação e controle

Este projeto tem como objetivo a construção de um carrinho seguidor de linha, utilizando um sensor de linha para detectar a trajetória e um motor para comandar o movimento do veículo. O controle é realizado através de um microcontrolador Arduino, no qual o sensor e o motor são programados utilizando a IDE do Arduino. O projeto integra conceitos de automação e controle, sendo parte integrante da disciplina do Projeto Integrado do curso de Análise e Desenvolvimento de Sistemas, sob orientação do professor Eduardo Furlan.

Especificações

Arduino Uno R3 (ou similar) com microcontrolador ATmega328P.

Ponte H Dupla L298N para controle dos motores.

Kit Chassi 2WD (2 rodas) Robô para Arduino.

Adaptador Bateria 9V para Arduino.

Sensor de Obstáculos Reflexivo Infravermelho.

Jumpers – Macho/Macho.

Jumpers – Macho/Fêmea.

Bateria 9V.

Pilhas AA.
